# Summer-Internship-2026

Merciya QuickWash - Local Laundry Booking System
Overview

Merciya QuickWash is a responsive front-end web application designed to simplify the laundry service booking process. The application enables customers to schedule laundry pickups, select services, make payments, track orders, manage profiles, and submit feedback through a user-friendly interface. It also includes an administrative dashboard for monitoring customer orders and business activities.

The project demonstrates the implementation of a complete service booking workflow using HTML, CSS, JavaScript, and browser Local Storage.
